# Monkey Jungle Jump (Chrome Extension)

A Chrome extension mini-game where you control a monkey:
- move left/right
- jump across water gaps
- avoid spikes
- collect coins
- collect extra-life hearts

## Install in Chrome

1. Open `chrome://extensions`.
2. Enable **Developer mode**.
3. Click **Load unpacked**.
4. Select `/Users/sara/Documents/New project/chrome-monkey-offline`.
5. Click the extension icon to open the game popup.

## Controls

- Keyboard: `A/D` or `Left/Right` to move, `W`/`Up`/`Space` to jump.
- Mouse/Touch: use the on-screen left, right, and jump buttons.
- Restart after win/game over: press `Space` or tap the canvas.
