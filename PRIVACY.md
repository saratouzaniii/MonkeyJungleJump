# Privacy Policy (Draft)

Monkey Runner does not collect, sell, or share personal data.

## What the extension uses

- `tabs`: detect tab loading state and open the local game page from the extension.
- `webNavigation`: detect top-level network navigation errors.
- `storage`: save user preferences (overlay enabled, offline redirect enabled, loading delay).

## Data handling

- Preferences are stored with Chrome extension storage.
- No analytics, no account system, no remote backend.
- No browsing history is transmitted to external servers.
